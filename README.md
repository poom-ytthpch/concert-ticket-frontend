## Project setup

```bash
$ git checkout main

$ cp example.env .env

$ npm install
```

## Compile and run the project

```bash
# watch mode
$ npm run sdev
```

Open [http://localhost:3000/your-path](http://localhost:3000/your-path) with your browser to see the result.
